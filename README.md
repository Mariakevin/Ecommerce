# Chennai Retail - E-Commerce

A premium e-commerce website template designed for Chennai-based retail businesses. Built with Tailwind CSS and Material Symbols, featuring a warm terracotta color palette and glassmorphism UI elements.

## Live Demo

[View Live Site](https://mariakevin.github.io/Ecommerce/)

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home page with hero, categories, featured products, newsletter |
| `category.html` | Category listing with filters, sort, and pagination |
| `Itempage.html` | Product detail page with images, reviews, and add-to-cart |
| `cart.html` | Shopping cart with order summary and recommendations |
| `checkout.html` | Multi-step checkout with delivery options and address selection |
| `orderconfirm.html` | Order confirmation with confetti animation |
| `myaccount.html` | User dashboard with orders, wishlist, and saved items |
| `searchresult.html` | Search results with instant suggestions and filters |

## Features

- Responsive design (mobile + desktop)
- Consistent navigation across all pages
- Glassmorphism UI components
- Material Symbols icon system
- Tailwind CSS with custom design tokens
- Smooth animations and transitions
- Dark mode support via class toggle

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Google Fonts (Montserrat + Inter)
- Material Symbols Outlined

## Color Palette

- **Primary:** `#9a442d` (Terracotta)

- **Secondary:** `#436086`
- **Surface:** `#faf8ff`
- **On Surface:** `#171b2b`

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Mariakevin/Ecommerce.git
   ```
2. Open `index.html` in your browser

No build tools required - all assets are loaded via CDN.
